[x] [Chrome browser]()


## Technical information
### Install dependencies
``````
npm install
``````
###

### Start the server
``````
npm run start
``````

### App site
- [LocalHost](http://localhost:3000)

### Kill process
``````
kill -9 $(lsof -t -i :3000) &
`````
